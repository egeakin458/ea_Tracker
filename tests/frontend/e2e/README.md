# Cypress End-to-End Tests

This directory holds end-to-end (E2E) smoke tests for the application using Cypress.

## Getting started

1. Install Cypress:
   ```bash
   cd frontend
   npm install --save-dev cypress
   ```

2. Open the Cypress test runner:
   ```bash
   cd frontend
   npx cypress open
   ```

3. Or run headless in CI:
   ```bash
   cd frontend
   npx cypress run
   ```
